Ice Cream Landing Page 🍦

Простой одностраничный сайт (landing page), созданный с использованием HTML5, предназначенный для презентации бренда мороженого, его продукции и мобильного приложения.

📌 Описание проекта

Проект представляет собой адаптивную HTML-страницу с несколькими логическими секциями:

навигация по странице

информация о компании

карточка продукта

описание мобильного приложения

форма обратной связи

футер с контактами и соцсетями

Сайт не использует JavaScript и CSS (в текущей версии), фокус сделан на семантическую структуру HTML.

Убедиться, что папка images находится рядом с index.html
